오행의 정의

오행은 다섯 가지 물질이 아니라
기운의 흐름과 작용 방식을 의미한다.

우주의 변화와 상호작용을 설명하는
에너지의 철학적 체계이다.

오행의 구성

  • 목(木) · 봄 · 솟아오름
  • 화(火) · 여름 · 확장
  • 토(土) · 환절기 · 아우름
  • 금(金) · 가을 · 수렴
  • 수(水) · 겨울 · 사라짐

목(木)

  • 기운 : 솟아오름 · 성장 · 진취
  • 계절 : 봄 / 아침 / 어린이
  • 품성 : 인(仁)
  • 포인트 : 탐색 · 공생 · 노화 방지
  • 맛 : 신맛
  • 식품 : 베리류 · 감귤 · 녹차
  • 신체 : 간 · 근육

화(火)

  • 기운 : 확장 · 발산 · 표현
  • 계절 : 여름 / 점심 / 청년
  • 품성 : 예(禮)
  • 포인트 : 자기표현 · 미디어 · 증명
  • 맛 : 쓴맛
  • 식품 : 커피 · 술 · 쓴나물
  • 신체 : 심장 · 혈관

토(土)

  • 기운 : 아우름 · 안정 · 중심
  • 계절 : 환절기 / 오후 / 중년
  • 품성 : 신(信)
  • 포인트 : 신뢰 · 기반 · 포용
  • 맛 : 단맛
  • 식품 : 쌀 · 고구마 · 꿀
  • 신체 : 위장 · 피부

금(金)

  • 기운 : 수렴 · 절제 · 제거
  • 계절 : 가을 / 저녁 / 장년
  • 품성 : 의(義)
  • 포인트 : 질서 · 약속 · 단절
  • 맛 : 매운맛
  • 식품 : 마늘 · 생강 · 무
  • 신체 : 폐 · 대장

수(水)

  • 기운 : 침잠 · 저장 · 통찰
  • 계절 : 겨울 / 밤 / 노년
  • 품성 : 지(智)
  • 포인트 : 정신 · 기억 · 생식
  • 맛 : 짠맛
  • 식품 : 해조류 · 소금
  • 신체 : 신장 · 뇌
