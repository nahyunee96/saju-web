표준시와 +30분 보정

한국 표준시는 UTC+9이나 실제 태양 고도 기준과 약 30분 차이가 발생한다.
따라서 사주 산출 시 +30분 보정을 적용한다.

천간과 지지의 체계

  • 천간 : 하늘 · 이상
  • 지지 : 땅 · 현실

지지는 복합 오행 구조이며,
이를 지장간이라 한다.

자(子)

  • 상징동물 : 쥐
  • 오행 : 수(水)
  • 시간 : 자시 (23:30 ~ 01:30)
  • 월 : 12월 (대설 ~ 소한)
  • 계절 : 한겨울
  • 지장간 : 계수 100 (본기)
  • 의미 : 모든 종자가 조용히 새끼치기를 시작하다
  • 괘상 : 지뢰복
  • 해석 포인트 : 시작 전의 응축, 잠재력, 저장

축(丑)

  • 상징동물 : 소
  • 오행 : 토(土)
  • 시간 : 축시 (01:30 ~ 03:30)
  • 월 : 1월 (소한 ~ 입춘)
  • 계절 : 늦겨울
  • 지장간 : 기토 50 + 계수 30 + 신금 20
  • 의미 : 땅으로 나올 준비를 하다
  • 괘상 : 지택림
  • 해석 포인트 : 응축, 인내, 준비

인(寅)

  • 상징동물 : 호랑이
  • 오행 : 목(木)
  • 시간 : 인시 (03:30 ~ 05:30)
  • 월 : 2월 (입춘 ~ 경칩)
  • 계절 : 초봄
  • 지장간 : 갑목 70 + 병화 30
  • 의미 : 하루를 시작하다
  • 괘상 : 지천태
  • 해석 포인트 : 개시, 추진, 기상

묘(卯)

  • 상징동물 : 토끼
  • 오행 : 목(木)
  • 시간 : 묘시 (05:30 ~ 07:30)
  • 월 : 3월 (경칩 ~ 청명)
  • 계절 : 완연한 봄
  • 지장간 : 을목 100
  • 의미 : 동쪽에 해가 뜨고 만물이 나오다
  • 괘상 : 뇌천대장
  • 해석 포인트 : 생장, 확산, 발아

진(辰)

  • 상징동물 : 용
  • 오행 : 토(土)
  • 시간 : 진시 (07:30 ~ 09:30)
  • 월 : 4월 (청명 ~ 입하)
  • 계절 : 늦봄
  • 지장간 : 무토 50 + 을목 30 + 계수 20
  • 의미 : 모든 만물이 활발히 움직이다
  • 괘상 : 택천쾌
  • 해석 포인트 : 전환, 활동성, 변화 직전

사(巳)

  • 상징동물 : 뱀
  • 오행 : 화(火)
  • 시간 : 사시 (09:30 ~ 11:30)
  • 월 : 5월 (입하 ~ 망종)
  • 계절 : 초여름
  • 지장간 : 병화 70 + 경금 30
  • 의미 : 기운이 맹렬히 상승하다
  • 괘상 : 중천건
  • 해석 포인트 : 추동, 폭발, 가속

오(午)

  • 상징동물 : 말
  • 오행 : 화(火)
  • 시간 : 오시 (11:30 ~ 13:30)
  • 월 : 6월 (망종 ~ 소서)
  • 계절 : 한여름
  • 지장간 : 정화 100
  • 의미 : 중천에 떠있는 밝은 대낮
  • 괘상 : 천풍구
  • 해석 포인트 : 극성, 정점, 노출

미(未)

  • 상징동물 : 양
  • 오행 : 토(土)
  • 시간 : 미시 (13:30 ~ 15:30)
  • 월 : 7월 (소서 ~ 입추)
  • 계절 : 늦여름
  • 지장간 : 기토 50 + 정화 30 + 을목 20
  • 의미 : 만물이 그 맛을 맞추다
  • 괘상 : 천산둔
  • 해석 포인트 : 조율, 수습, 완화

신(申)

  • 상징동물 : 원숭이
  • 오행 : 금(金)
  • 시간 : 신시 (15:30 ~ 17:30)
  • 월 : 8월 (입추 ~ 백로)
  • 계절 : 가을
  • 지장간 : 경금 70 + 임수 30
  • 의미 : 만물이 활짝 펴다
  • 괘상 : 천지비
  • 해석 포인트 : 결실 직전, 분리

유(酉)

  • 상징동물 : 닭
  • 오행 : 금(金)

  • 월 : 9월 (백로 ~ 한로)
  • 계절 : 가을
  • 지장간 : 신금 100
  • 의미 : 수확과 정리
  • 괘상 : 풍지관
  • 해석 포인트 : 완성, 판단, 평가

술(戌)

  • 해석 포인트 : 탈락, 정리, 종료

해(亥)

  • 상징동물 : 돼지
  • 오행 : 수(水)
  • 시간 : 해시 (21:30 ~ 23:30)
  • 월 : 11월 (입동 ~ 대설)
  • 계절 : 겨울
  • 지장간 : 임수 70 + 갑목 30
  • 의미 : 씨앗이 다시 생기다
  • 괘상 : 중지곤
  • 해석 포인트 : 순환, 재시작, 회복

지장간의 이해

  • 계절의 첫 지지 : 다음 계절 준비
  • 가운데 지지 : 순수 오행
  • 마지막 지지 : 토가 마무리

지장간의 발견으로 사주 해석은
고법에서 자평법 중심의 신법으로 발전하였다.
