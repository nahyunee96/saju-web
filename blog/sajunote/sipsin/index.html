사주 기초

십신(十神) — 비·식·재·관·인

십신을 “삶/길/입출력/경계/규칙”으로 매핑해서 이해하는 정리본. (삶=내가 걸어가야 하는 길, 사주=그 길의 환경과 내 특성)

🧭 큰 비유로 정리

삶
내가 걸어가야 하는 길
사주
내가 걸어갈 길의 환경과 나의 특성
비(비겁)
나 자신의 기운 = 나의 머리, 몸
식(식상)
아웃풋 = 나의 손, 입, 재주
재(재성)
나의 발, 다리, 발걸음 / 영향력 / 바운더리
관(관성)
척추(기준, 움직임의 축) / 골격 / 신경계(통제) · 질서 · 구조 · 울타리 · 신호등
인(인성)
인풋 = 나의 눈, 귀

읽는 순서 팁: 비(나)가 움직이고 → 식(표현/기술)로 결과를 만들고 → 재(확장/돈/영향)이 생기고 → 관(규칙/통제)이 구조를 잡고 → 인(학습/보호)가 입력과 기반을 채운다.

비견 겁재 식신 상관 편재 정재 편관 정관 편인 정인
비(比)

비견(比肩)

“어깨를 나란히” — 나와 같은 기운이 옆에 서는 형상

펼치기
주체성 독립성 자존심 승부욕 동료/협업
기본 정의
比肩 = 같은 자리·같은 힘으로 어깨를 맞대다. 곧 “나 자신과 똑같은 기운”이 일간 옆에 서는 형상. 범위: 일간(나)과 오행·음양이 모두 같은 기운.
핵심 키워드
주체성, 독립성, 자존심, 승부욕, 목표의식, 동료·협업
심리/행동
독립·자유인. 간섭·명령을 싫어함. “내 길은 내가 정한다.” 생각 → 즉시 실행, 용맹·추진력, 승부를 즐김.
관계/재물
관계: 형제·동료·팬덤처럼 ‘평등·동맹’ 지향 ↔ 과하면 고독·갈등. 재물: 편재(돈)를 제어·분배 ⇒ “내 손으로 직접 벌고 쓰겠다.”
물상
기둥·탑·건강기구·체육시설·전주(電柱)처럼 스스로 선 구조물

👍 긍정

  • 강한 의지·자존, 정정당당한 경쟁으로 목표 달성
  • 비견 하나는 리더·프리랜서 에너지
  • 목표를 명확히 세우고 독립 프로젝트를 추진하면 상승

⚠️ 부정

  • 독불장군·고집, 타협 불가
  • 투자/의사결정에서 “내가 맞다”에 과몰입하면 실패 위험
  • 결정 전에 “내가 진짜 하고 싶은 일인가?” 자문 필요

★ 개수별 체크

  • 적당: 주체성 확보 + 동료와 협업 가능
  • 과다: 외부 경쟁 없으면 고립/사회성 저하 → 협업 구조로 분산
  • 부재: 주체성 부족·종속적 삶 → 비견 운 때 자립 기회 잡기

타 십신과의 관계

  • 극받음(편관): 비견을 ‘제도·규율·상사’가 누르는 힘 → 고집 관리
  • 극함(편재): 돈·자원 분배 갈등 → 공정 분배 규칙 선제 설정
  • 동류(겁재): 순수 경쟁(비견) ↔ 공격적 경쟁(겁재) 경계선 점검

직업·생활 적용 / 운

  • 직업: 프리랜서·의사·변호사·언론인 / 스포츠 선수·직업군인·경호원
  • 전략: 목표·가치 설정 → 실행 속도 극대화 / 동급 파트너와 콜라보 / 자본·시장은 투명 규칙으로 관리
  • 운에서 비견: 플러스(자립/동업/성취) · 마이너스(갈등/몫 싸움) → “협력인가 소모전인가?”

비견은 ‘나와 똑같은 힘’이 옆에 서며 순수한 주체성·승부욕을 증폭시킨다. 잘 쓰면 독립·리더십·정정당당 승부, 과하면 고집·고립. 목표 → 규칙/협업 구조 → 피드백 루틴이면 사회적 성취로 연결됨.

비(劫)

겁재(劫財)

“빼앗아서라도 지킨다” — 생존본능·경계심·제로섬 승부

펼치기
자주성 경계심 생존본능 경쟁심 레버리지
기본 정의
劫 = 강탈, 財 = 재물 → “내 몫을 지키기 위해 상대의 것을 빼앗는 힘” 범위: 오행은 같고 음양만 다른 기운(甲↔乙, 庚↔辛 등)
핵심 키워드
자주성, 경계심, 생존본능, 경쟁심, 질투, 전략적 약탈
심리/행동
“뺏기면 죽는다” 생존 의식. 배신·질투에 민감. 불가능을 뚫는 오기·돌파력, 때론 편법·강행.
관계/재물
관계: 형제·친구 = 경쟁자. 필요하면 협력/배신도 불사. 재물: 정재(안정자산)를 흔들어 유동·레버리지화 → 패재 위험도 동반.
물상
동굴·근육·서까래·운동기구처럼 “몸과 터를 지키는 무기”

👍 긍정

  • 극한 상황 돌파, 큰 승부에서 한방 성취
  • 위험/보상을 계산해 “한정판 스퍼트”로 쓰면 강함

⚠️ 부정

  • 질투·약탈·소송, 투자 과열로 패재(敗財)
  • 룰 없이 쓰면 인간관계와 돈이 같이 터짐

★ 개수별 체크

  • 적당: 위기 돌파용 비상 엔진
  • 과다: 상시 전투 모드 → 정재 고갈·인맥 손상 → 정관으로 절제 필수
  • 부재: 경쟁 회피·의존 심화 → 겁재 운에 각성 기회

타 십신과의 관계

  • 극받음(정관): 규칙·법·상사가 제동 → 폭주 방지
  • 극함(정재): 잠자는 자본을 깨워 유통/투자 활성 → 회계/분배 투명화
  • 동류(비견): 정정당당 vs 편법 경계선 확립

직업·생활 적용 / 운

  • 직업: 벤처 CEO·투자 트레이더·M&A / 특수부대·경호·격투 스포츠
  • 전략: 제로섬을 동맹 구조로 바꾸기 / 정관·규약을 미리 명문화 / 승부 전 “정말 빼앗을 가치가 있나?” 자문
  • 운에서 겁재: 플러스(조력/투자/위기 뒤 수익) · 마이너스(배신/소송/몫 탈취) → “협업 확대인가, 약탈 소모전인가?”

겁재는 “살아남기 위해 빼앗고 쟁취하는 힘”. 잘 쓰면 위기 해결사, 과하면 패재. 룰로 묶고 → 확장 게임으로 전환 → 필요 순간만 폭발이 핵심.

식(食)

식신(食神)

“먹고 길러 내는 힘” — 창조·배양·생산·건강

펼치기
식복 창의력 탐구심 생산성 건강 배려
기본 정의
食 = 먹다·養, 神 = 작용의 주체 → “양분을 받아 창조·배양하는 에너지” 범위: 일간이 생(生)하는 같은 음양의 오행의 기운 (예: 갑木 → 병火)
심리/행동
현실을 즐기는 낙천성. 몰두·집중으로 “한 우물” 파기. 꾸준한 실행력·제조력, 손발(도구)을 써서 결과물을 만든다.
관계/재물
관계: 소탈·친화적 커뮤니티, 돌봄·교육을 통한 성장 지원. 재물: 편재를 부드럽게 길러 분배 → “내 손으로 생산해 먹고 산다”.
물상
부엌·연구실·농장·공방·도구·식물처럼 “키워 내는 공간/도구”

👍 긍정

  • 건강·장수, 창의적 연구·제조, 차분한 설득력
  • “하나만 깊게 파자” 전문성 집중 전략에 강함

⚠️ 부정

  • 게으름·무사태평, 소비 과다
  • 공상만 하고 실행 부족 → 루틴/마감이 필수

★ 개수별 체크

  • 과다: 욕망·표현은 큰데 실행 지연 → 편인으로 정리/이론화 필요
  • 부재: 실속·전문성 부족, 직장 인연 약화 → 식신 운에 기술/취미 배양

타 십신과의 관계

  • 극함(편관): 규율을 부드럽게 제어해 활로를 냄(완충 역할)
  • 극받음(편인): 연구/이론 > 실행으로 치우침 → 실천 일지로 행동 비율 유지
  • 동류(상관): 식신(꾸준) ↔ 상관(폭발) → 번갈아 써야 “꾸준함+임팩트”

직업·생활 적용 / 운

  • 직업: 연구원·기획자·작가·교사·의사·기술자 / 요식업·제조업·농산업·호텔·도매업 / 보육·복지
  • 전략: “가장 즐겁게 몰두할 단 하나”를 정해 깊게 / 느긋한 리듬 + 마감으로 스위치 / 소규모 커뮤니티로 확장
  • 운에서 식신: 플러스(취미·기술·생산 자원) · 마이너스(소비·게으름) → “성장 투자인가, 소모성 낙비인가?”

식신은 “양분을 받아 창조·배양하는 힘”. 전문 목표 + 실행 루틴 + 소규모 연대가 있으면 현실 성과 + 삶의 여유로 이어진다.

식(傷)

상관(傷官)

“규칙을 깨고 새길을 여는 칼” — 표현·변화·창조적 파괴

펼치기
표현력 사교성 창조적파괴 교섭력 센스 변화추구
기본 정의
傷官은 ‘관(정관)을 상하게 한다’는 뜻 → 통제·관습을 벗기고 표현·변화를 촉발하는 날카로운 힘. 범위: 일간이 생(生)한 오행 가운데 음양이 다른 기운 (예: 갑木 → 정火)
심리/행동
자유분방·호기심. “틀을 깨야 빛난다.” 화려한 언변·임기응변, 100을 10000으로 부풀리는 퍼포먼스.
관계/재물
관계: 폭넓은 친교·네트워크, 다만 허세·충동으로 갈등도 잦음. 재물: 정재를 흔들어 ‘새 돈길’ 개척 → 창업·유통·콘텐츠 수익화.
물상
마이크·무대·언론·유흥 시설·명품 — 화려함과 즉흥성을 담은 도구

👍 긍정

  • 외교·설득·재능 발현, 고정관념 깨는 혁신
  • “새로움 + 임팩트”를 명확한 목표에 집중하면 흥행

⚠️ 부정

  • 허영·과시·규범 위반
  • 직장 인연 단절·소송 위험 → 룰/마감/코치로 탈선 폭 제한

★ 개수별 체크

  • 적당: 재능을 적소에 빛내는 변화 촉진자
  • 과다: 과잉 표현·직업 변동 ↑ → 정인(논리)·정관(규칙) 필요
  • 부재: 표현력·변화 대응 약함 → 상관 운에 PR/창의 훈련

타 십신과의 관계

  • 극함(정관): 권위/규칙을 깨 혁신 유발(과하면 무질서)
  • 극받음(정인): 지식·이론화로 눌림(아이디어만 남지 않게 실행 비율 유지)
  • 동류(식신): 식신 루틴 속 상관 쇼타임 구간 확보

직업·생활 적용 / 운

  • 직업: 아나운서·연설가·변호사·정치인·MC / 세일즈·외교·무역·유통 / 배우·가수·디자이너·IT 스타트업
  • 전략: 무대 설계(SNS/브랜딩) + 룰 세팅(정관/마감/계약) + 정인 피드백
  • 운에서 상관: 플러스(재능 부활/친교 확대/창업) · 마이너스(구설/소송/허세 투자) → “쇼타임인가, 사고칠 타이밍인가?”

상관은 “틀을 깨고 표현·변화를 촉진하는 칼”. 빛날 무대 → 명확한 룰 → 이론 피드백 3단계면 혁신과 흥행으로 연결된다.

재(偏)

편재(偏財)

“큰 판돈을 굴리며 세상을 연결” — 확장·네트워크·투자

펼치기
확장성 네트워크 투자 판단속도 모험 낙천성
기본 정의
偏 = 치우치다, 財 = 재물 → 내가 소유하기보다 운용·확장하는 ‘큰 돈’. 범위: 일간이 극(剋)하는 오행 가운데 음양이 같은 기운 (예: 갑木→무土, 병火→경金)
심리/행동
낙관·스피드 우선, “보유보다 활용” 마인드. 대담한 베팅·빠른 실행, 공간 감각·판단력.
관계/재물
관계: 넓고 얕은 인맥·파티·연예 기질, 연결자 역할. 재물: 공공재·투자금·대출금까지 굴려 큰 현금 흐름 창출.
물상
지도·네트워크망·공항 관제·대형 매장 등 “넓은 무대” 도구

👍 긍정

  • 시야 넓고 기회 포착 빠름
  • 큰 자본·인맥을 모아 확장 성공

⚠️ 부정

  • 조바심·투기 중독, 수입 불규칙
  • 리스크 과대 → 손절 규칙 + 재무 안전망(정재·정관) 필요

  • 전략: 학위/자격/논문/특허 갱신(지식 포트폴리오) / 분기별 실전 프로젝트로 이론→성과 / 멘토-제자 라인 구축
  • 운에서 정인: 플러스(학위·자격·부모/스승 도움·유산) · 마이너스(안주·실행 저하) → “근본 강화인가, 현실 도피인가?”

정인은 “보편적 원리와 보호로 가치를 영구히 남기는 인장”. 원리 확립 → 현실 적용 → 지식 전승이면 세대를 잇는 가치와 풍요로 이어진다.
